# Cars Crash (Python + Pygame)

## Overview

Cars Crash is a simple arcade-style car racing game developed in Python using the Pygame library. The player controls a car and must avoid incoming enemy vehicles while earning points. The game includes collision detection, score tracking, increasing difficulty, and restart functionality.

## Features

* Player Car Movement
* Enemy Vehicle System
* Collision Detection
* Score Tracking
* Increasing Difficulty
* Game Over Screen
* Restart Functionality
* Smooth Gameplay

## Technologies Used

* Python 3.12
* Pygame

## Controls

* Left Arrow → Move Left
* Right Arrow → Move Right
* Enter → Restart Game

## Installation

### Install Pygame

```bash
pip install pygame
```

### Run the Game

```bash
python cars_crash.py
```
